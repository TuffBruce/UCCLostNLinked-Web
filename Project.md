## Pages:
- Public pages:
    - App overview page
    - Login page
    - Signup page
- Protected pages (must be logged in):
    - Home page (display found items and lost items)
    - Upload found item (Form)
    - Report lost item (Form)
    - Gradtitude page
    - Optional: Admin page (only visible for an admin user)

Protected pages can only be viewed if authenticated (user is logged in). If not, they will redirect the user to the login page.
